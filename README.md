# 🎮 J4U Arcade

**J4U Arcade** is a desktop arcade game application built with **Java Swing**.  
The project brings together **8 small mini-games** into a single program, allowing players to choose and enjoy different games from one main interface.

This project was created for learning and practicing **Java GUI programming**, **object-oriented design**, and **event-driven programming** using Swing.

---

## 🕹️ Project Overview

J4U Arcade simulates a simple arcade experience where users can:
- Launch the application from a main window
- Select one of the available mini-games
- Play the game inside the same application
- Return to the main menu and choose another game

The focus of this project is not only gameplay, but also:
- Managing multiple screens
- Handling user interactions
- Organizing a larger Java project with multiple classes

---

## 🎯 Features

- 🧩 **8 Mini Games** included in one application  
- 🖥️ **Graphical User Interface** built with Java Swing  
- 🔁 Smooth screen switching between menu and games  
- 🧠 Uses **Object-Oriented Programming (OOP)** principles  
- 🎮 Simple and beginner-friendly gameplay  
- 💡 Designed for learning and experimentation

---

## 🛠️ Technologies Used

- **Language:** Java  
- **GUI Framework:** Java Swing  
- **IDE:** IntelliJ IDEA / Eclipse / VS Code
- **Paradigm:** Object-Oriented Programming  
- **Event Handling:** ActionListener, KeyListener
